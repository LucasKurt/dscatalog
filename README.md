# dscatalog
Projeto de API rest feita em java com spring do bootcamp da DevSuperior pelo professor Nelio Alves

## Backend

### Tecnologias
  - SpringBoot
  - Spring Web
  - Spring data JPA
  - String validation
  - Oauth2

### Funcionalidades
  - Crud Completo (Get all paginado, get one, post, put , delete)
  - Testes automatizados
  - Validação
  - Autenticação
